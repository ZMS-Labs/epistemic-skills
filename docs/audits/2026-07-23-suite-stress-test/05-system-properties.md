# 05 — System properties

## Method and result vocabulary

This report tests the suite-level properties named by OUT-005 against the packet commit `9532a57199fc8d4747a91916d59d1ea86c34d838`. A property is not marked sound merely because a unit test passes.

- **SOUND** — direct source plus deterministic or live evidence supports the stated scope.
- **CONDITIONAL** — the contract is coherent and some evidence exists, but a required runtime, independence, or real-world measurement is absent.
- **OPEN** — the expected property could not be established in this target.
- **CONTRADICTED** — observed behavior conflicts with the property.

## System-level conclusion

The suite's envelope/truth distinction, negative-trigger discipline, repo-backed relay, deterministic contracts, and authority boundaries held in this audit. The correct overall status is therefore `PARTIAL`, not a rounded-up pass.
