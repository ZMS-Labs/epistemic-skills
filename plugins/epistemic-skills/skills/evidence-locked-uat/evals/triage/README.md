# UAT proportionality triage fixtures

This deterministic battery tests the boundary before an acceptance run starts. It does
not claim to execute UAT.

- A routine, reversible presentation change gets exactly the bounded five-line check and
  no UAT identity, packet, manifest, hash chain, role call, or verdict.
- Stateful keyboard/focus behavior and an explicit operator UAT request enter the
  independent UAT pipeline.
- A material acceptance target that cannot be reached is `BLOCKED_ENVIRONMENT`, never
  `PASS` and never downgraded to source inspection.

Run `python tests/run_tests.py`.
